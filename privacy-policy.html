Privacy Policy

Last Updated: January 9, 2025

Overview

Agent Hub Document Collector is an internal productivity tool for collecting Notion and Confluence pages. This extension is designed for enterprise use only.

Information We Collect

This extension collects the following information to provide its core functionality:

  • User email address for authentication
  • Page URLs and titles from Notion and Confluence
  • Document collection queue data
  • User preferences and settings

How We Use Information

The collected information is used solely for:

  • Enabling document collection and upload functionality
  • User authentication and authorization
  • Maintaining collection queues and deployment status

Data Storage

Data is stored in two locations:

  • Local browser storage (automatically deleted when extension is removed)
  • Internal company servers (accessible only to authorized users)

Data Sharing

We do not share, sell, or distribute any collected data to third parties. All data remains within the internal company system.

Security

All data transmission is encrypted using HTTPS. Access is restricted to authenticated company employees only.

Your Rights

You can delete all local data by removing this extension from your browser. For server data deletion, please contact us at the email below.
